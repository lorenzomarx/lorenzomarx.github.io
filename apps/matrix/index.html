Matrix — Step Sequencer

Click cells on the grid to lay down notes. Each row is a different pitch (C major pentatonic, so no wrong notes). Each column is a 16th-note step. The playhead loops left to right.

Controls

Click
Toggle a cell on or off
Drag
Paint multiple cells at once
Play
Start / stop the loop
Space
Same as Play
Tempo
Slide between 40 and 240 BPM
Clear
Wipe the grid
Share
Copy a URL that contains your pattern

About

Inspired by Andre Michelle’s ToneMatrix and lupine-dev’s ToneMatrix Redux. Built from scratch in vanilla JS + Web Audio for this site.
